Platform Developer Advanced Topics
=======================================

.. _advanced-developer:

This section covers advanced topics useful to Platform Developers. A Platform Developer is a programmer that intends to modify the source code of the ODK 2 tools themselves. This person might want to add a new view type or a fix a bug.

Other perspective definitions can be found :ref:`here <odk-2-perspectives>`.
